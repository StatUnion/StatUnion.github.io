求职大视野数据分析

求职大视野 | 数据分析秋招拿了5个offer，记录我的艰苦历程

2021-02-16
数据分析, 求职大视野
本文总阅读量次

基本情况

本硕双非，一个外企市场部数据分析实习。研一研二浪的太欢了，找实习的时候被拒到自闭，认清了自己菜狗本质，开始面向工作学习，利用暑期两个月自学了SQL、Python、数据结构、机器学习（这两个月是我整个学生生涯最认真的时候，别人都在实习，我天天跑机房恶补，出来混总要还的/(ㄒoㄒ)/~~）

整体情况

基本都是数据分析岗，投递 70+，面试 11家，offer 5个

上海银行（一面挂）

面试完没及时记录，现在想不起来问了啥。面完就知道自己凉了，一直追着问我数学建模细节，我忘的差不多了，支支吾吾的超级惨，所以写在简历上的一定要好好准备，不然就是自己给自己挖坑。

58

（一面完没消息了，官网一直待面试。。。）

  1. 自我介绍
  2. 介绍一下你实习的主要内容
  3. 你实习中哪些项目是你主要负责的，主要成就是什么？
  4. 实习给你带来的收获有哪些?
  5. 你有参加两个竞赛，选择其中一个详细介绍一下
  6. 你在团队中的角色?你们思考解决问题的逻辑是什么？
  7. 你觉得你从事数据分析岗位的优势?未来发展规划

反问：

  1. 公司的数据分析岗是偏业务还是偏技术?
  2. 应届生培养机制?

平安产险（offer）：

一面（HR面，1V4，1是面试官）：

  1. 自我介绍
  2. 三个词形容自己
  3. 最近影响最深的一次活动？为什么？
  4. 为什么会想到跨专业考研？

终面（3V1,我是那个1）：

  1. 自我介绍
  2. 我提到了期待养老这款保险，然后面试官问我希望这个产品偏服务，还是偏基金？希望这个产品还有什么其他惊喜的点（我说社区，社交，老了会有孤独感）
  3. 对国内镍了解吗（不了解我主要做的是铁矿分析）？
  4. 做了那么久的实习？说一个你最大的收获？
  5. 职业规划？互联网＋是个很大的方面，你未来主要想从事哪个方面？

反问：

  1. 是否还有其他面？（这是终面，我问了我是否能得到这个offer(面试官说他会给我发的)然后问我能不能来实习，我说是先实习再考虑是否能转正的吗？不是先发offer吗？）
  2. 薪资构成，应届生调薪的周期？
  3. 可以介绍一下未来大致的工作内容吗？

神策数据（offer）：

一面（群面，3V6，3个面试官）：3个人一组，40分钟两道题小组无领导讨论，最终汇报，汇报完面试官和另一小组成员都可以进行提问。我们组抽到的两个问题：

  1. 估算今年国庆全国的旅游消费总支出？
  2. 活动运营策划，为提高星巴克app星推送模块的复购率设计运营活动，并设定三个主要的评价指标？

二面（3V2，3个面试官）：

  1. 对数据分析岗位的理解？
  2. 如果你手头有个项目时间可能刚刚好在截止日期能完成，但是突然有个项目***来要求本周完成，你如何平衡？
  3. 结合你实习中的业务场景，介绍一下获取和处理数据的途径和流程？
  4. 对神策的了解？
  5. 加入神策你希望收获什么？

反问：

  1. 神策目前公司规模，有哪几个分公司？上海分公司业务辐射范围？
  2. 可以介绍一下公司的主要业务流程吗？主要服务的公司类型？
  3. 应届生的培养计划？
  4. 加班情况？

三面（1V1,主要就是聊天）：

  1. 你的朋友对你的评价，用两个词概括？
  2. 你觉得你朋友对你的评价和你自己对自己的认知有没有出入？
  3. 你秋招手里拿了几个offer？会优先选择神策吗？
  4. 说一件你觉得成就最大的一件事？

四面（1V1，还是聊天）：

  1. 未来职业规划?秋招主要投递哪些类型的公司？岗位有哪些？
  2. 原生家庭情况？对你的影响？
  3. 你的研究生经历和实习给你带来了什么？
  4. 你觉得从事你应聘的岗位你的优势在哪里？
  5. 你选择神策的理由？
  6. 为什么选择跨专业考研？导师是自己选择的吗？这个研究方向是你喜欢的吗？你的收获有哪些？

苏宁（一面挂，1V1）：

  1. 自我介绍
  2. 介绍一个你最熟悉的项目
  3. 为什么会想到用图卷积来解决这个问题？ 有没有想过用传统的机器学习方法来解决你这个问题？（我自己的项目）
  4. L1、L2的原理？两者区别？
  5. 偏差和方差的区别？
  6. 损失函数优化？你知道有哪几种？Adam可以讲一下吗？
  7. 了解决策树吗？可以讲一下决策树的原理吗？

/(ㄒoㄒ)/~~面试过程中我说的最多的一句话就是 “不好意思，这一块我不是很了解”。面的我快哭了，真的是菜啊。

浦发银行

（3V1，没收到体检通知，应该凉了)我是第一个，约的8：30，快9点半了才面试，陪他们调试了好久的设备。。。

  1. 3选1即兴演讲，我抽到的题目是：举例说明科技如何降低人的自主性和独立性
  2. 自我介绍
  3. 介绍一下k-means,你的数据如何处理，模型的输出是什么？（我自己简历里面的项目）
  4. SQL中的聚合函数有哪些？如何查询第一行？（我第一次听说SQL里还有聚合函数。。。。之前都是直接刷的SQL大题，哭了）
  5. 除了SQL还有使用过其他什么数据库吗？
  6. 实习中用到的数据分析软件？Power BI和Excel的对比？
  7. 你在数学建模竞赛中主要负责什么工作？
  8. 大数据分析方法与应用这门课主要讲什么？
  9. 如何确保客户数据的安全？数据中台？（我这边答偏了，刚好前一天浦发盗用别人身份贷款上了热搜，然后我作死提了，又是熟悉的凉凉节奏）

中银金科

（3V1，面完第二天就收到了体检通知，效率挺高的）吐槽一下中银家的笔试，简直不是给人做的，前面还看一下题目，后面真的连看完题目的时间都没有，全部闭眼选C。

  1. 自我介绍
  2. 你在那个公司实习一年多为什么不转正？研一就实习了如何平衡学业？（吹了一波工作效率高，学习能力强，其实我就是穷，划水实习赚生活费的🤣）
  3. SQL熟悉吗？用过什么SQL软件？你的数据量是多少？（我参加的一个数据分析比赛）
  4. 你的钢厂配矿数据库是怎么做的？为什么这样做？(实习内容）
  5. 你数学建模用的什么语言？主要用到哪些包？缺失值如何处理的？

南京银行（offer）：

一面(群面12人，三个面试官，其中一个cue流程)

  1. 讨论一个题目，先不按顺序每人发表一分钟
  2. 团体讨论15分钟
  3. 最后选出一个代表，总结陈述，组内其他人员可以补充。

(我们组进了6个，应该是刷一半的)

二面(4v1，我是那个1)：

  1. 一分钟自我介绍，并说明自己的优势
  2. 列举一个你团队合作中，如何合作解决困难的
  3. 你的缺点是什么?
  4. 你秋招都投递什么岗位?
  5. 你的研究生研究方向和实习经历和金融差的比较远，有补充过金融方面的知识吗？(因为我说我都投递的银行数分挖掘岗)
  6. 你有了解数据分析领域的各种考试和证书吗？
  7. 对产品经理的岗位感兴趣吗？（我投递的是数据分析挖掘岗）

(应该还有其他的问题，记得不大清楚了，前一天晚上熬夜付尾款，没咋睡觉，然后第二天面试的时候脑子是凌乱的)

大华（一面挂）：

我投的明明是产品经理岗，然后突然电话过来面试，说我过了他们那边HR面（我感觉我没面过🤣），现在进行技术面。给我转的海外经理岗，英语面试尬的我已经抠出三室一厅了，我英语是真的垃圾。

建行上海分行

（4V2，没收到后续应该也凉了）
我跟英语杠上了，睡得迷迷糊糊的下午1点开始面试，一进面试间就让我用英语介绍自己？？？我错了上一次说要准备英语自我介绍的时候就应该立马准备的，一直拖拖拖，我秋招结束了还是没准备。。。

  1. 用英文说一下你为什么应聘这个岗位？你的优势在哪里？你想通过职位提升你的什么技能？(黑人问好，为啥金融科技岗对英语要求那么高？又是凉的节奏）
  2. 介绍一下你在银行实习的内容（还是我本科的时候实习过）
  3. 有跟上级反映过这个问题吗？最后问题如何解决的？（我上面提到的银行营销不够精准）
  4. 你在银行实习过吗？实习中有发现需要改进的地方吗？（问另一个小姐姐的，因为我上面的回答，感觉她被我带沟里了，对不起小姐姐）
    还有一个研究所的数据分析岗的offer，但是在上海，我穷没法去养老了。

PS：

  1. 外地高校去南京面试可以申请南京政府1000块钱的面试补贴（需要面试公司提供面试证明）
  2. 要投递一家公司就立马投递，不要想着明天再投以后再投，这种情况很可能最后就变成了意念投递
