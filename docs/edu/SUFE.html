    # 小统侃院校

    上财，位于魔都杨浦区五角场武川路，顶级财经院校，与上海金融中心的名号相倚相成，知名度甚高，校友会在上海极具影响力。其统计学专业成立于1946年，积淀颇深，大师云集，15年全国应用统计专硕评估位居第一，17年统计学科入选世界一流学科建设名单。有两个学院招收应统专业学生，分别是统计与管理学院和数学学院。

    统计与管理学院在上财是既神秘又特殊的存在，学风扎实，大神云集，能文能武，低调内敛。虽然看似与金融、经济不沾边，但就业和其它任何专业相比从来没输过，薪资水平甚至更甚一筹。其特殊之处在于，它是上财为数不多的偏向于理工科的一个专业。上财应统有AB组之分，20年后之前的AB组统一为A组（另增上财-波士顿项目为B组），分为四个方向，依托于其深厚的财经背景与强大的财经师资力量，有其得天独厚的优势，课程安排集中在经济与金融统计方向：

    1. 数据科学与商务统计
    2. 人工智能与金融统计
    3. 机器学习与健康统计
    4. 大数据技术与经济统计

    此外21年统院新开设了一个上财-波士顿计划，也就是现在打的B组，该计划在时间、学费以及研究方向上与A组有所区别，是一个国际合作项目，其中1.5年在本校学习，1.5年在波士顿大学学习，学费标准为人民币20w元+美方8.2W刀，美方生活费自理，基本上决定只有少部分考生具备报考条件，所以应该是最好考的。因为学费巨高，21年没有报满，本来划线351，因为生源不足申请降到国家线348。

    看一下每年的招生情况，21年分数线379分，录取68人；20年分数线362，报考428人，录取66人；19年A组分数线376，报考156人，录取17人，B组382分，报考269人，录取34人；18年A组报考217人，录取32人，分数线360，B组报考116人，录取18人，分数线363分；17年365人，录取46人，分数线为404分；16年报考257人，A组录取26人，B组录取4人，分数线372分。每年的报录比保持在8：1左右。

    数学学院的应用统计刚开设一年，方向是是统计计算与数据建模，因为是第一年，分数线也不高，21年分数线为351分，进入复试23人，录取了17人。

    上财的专业课题型非常多变，有点让人捉摸不透，可以看下每一年的题型：

    年份 题型
    2020年 单项选择、简答、计算、应用分析
    2019年 单项选择、计算、应用分析
    2018年 单项选择、计算、应用分析
    2017年 单项选择、计算、证明与计算

    上财专业课试题每年都会加入新元素，历年的难题会默认为当年的基础题，当年又会加入新的难题。比如20年选择题加了很多时间序列分析的内容，最后一大题也是。所以把历年真题都学会是一定一定要的，不然肯定考不了高分。

    首先从题型分布看，专业课的题型按照往年初试的模式分为40分的选择题，10分的简单，80分的计算大题和以及具有一定超纲内容的20分建模大题这四种。总的考试时间为3个小时，但是正常完成卷子主体部分的时间大约在2小时。剩下将会有比较充裕的时间进行查漏补缺，以及验算自己有疑问的大题。对于可能存在不会的部分，可以适当按照自己所学的内容，填写相应的部分内容。尽可能的不要出现空着不填这种情况。

    第一部分的选择题较为基础，多为基本概念的灵活辨析，完成的速度一般都较快，但是需要建立在较高的正确率上。总体的出题顺序基本按照茆书的章节顺序由简单到困难进行递进，这些选择题的正确率和做题速度，都可以在平时通过完成其他学校432的真题加以训练。

    第二部分的简答题，要更加关注比较每一个章节关键的名词概念，对于茆书每一章所涉及到的名词建议都有所记忆，答题时可以灵活作答，若对某一概念没有较为准确的说法，可以使用场景问题进行解答，但最好要有准确定义。这一部分完全都是可以在考前进行充分准备的内容，这一部分建议同学们在考前最后的一段时间内自己抽出一定的时间进行归纳，系统性的归纳将有助于提升简答和选择的理解。

    第三部分的计算题，这一部分难度呈现逐步递进的趋势。在前几道题的难度较为简单，主要涉及到基本的概率密度的计算。对于这一部分的回答需要尽可能的不要省略步骤，并确保答案正确，完整的解答步骤能保证最后获得一定的步骤分。在数理统计的考核内容部分会涉及证明和计算，证明题可能是茆书中较为经典的那几个，并在一定程度上加以衍生，因此对于书上的证明内容要熟练推导。在答题过程中，如果难以进行完整的证明推导，也一定不要空着，可以适当的写一些书上有的相关的证明步骤。

    计算题可分为概率论部分和数理统计部分，题型可参照茆书第3章和第6、7章，这一部分用时建议在40分钟以内。同时值得注意的是，在这一部分也会出现概念的灵活应用，可能会出现一些新的概念，若第一遍答题过程中答不出建议暂时跳过，等到检查过程中进行回答。这里值得注意的一点是，由于后面几道简答的运算步骤都较为复杂，因此一定要将核心的步骤放在较为显眼的位置，同时要在小题的最后，对结论部分进行总结。常见的总结包括，假设检验的结论，回归分析模型的含义（注意几个回归系数平均含义的准确表达，方差分析模型的结论书写，参数估计结论中，对于显著性水平的描述）。以上的列举都会影响到最终的得分，因此千万注意！

    超纲的模型大题，这一部分一般由1-2小题，考核内容出现过时间序列，方差分析，回归建模以及部分更加深入的统计方法等等，总体更加考察考生的综合统计素养。一般而言是从头到尾的独立完成，即从原假设与备择假设，统计量，拒绝域等均由自己进行叙述与计算，这一部分建议时间充裕的同学可以多进行一些准备，时间不太够的同学也必须将基础的单因素方差分析，最小二乘法求时间序列掌握，而且由于难度较高，在做这一部分题可以评估一下自己当前剩余的时间和自己是否有确切的思路。对于想要取得较高分数的同学，建议先对题目所考察的内容进行初步判断，再将自己学过的能写上去的属于这一块的都尽可能的填写，万一有那么一些建模步骤能获得一些分数，都是非常赚的。

    关于参考资料，有两本主要参考的资料，一本是贾俊平的《统计学》，另一本是茆诗松的《概率论与数理统计》，当然如果可以广泛的参考一些其他资料夯实一下就更好了，因为上财的专业课也是有偏向于于统计计算证明的趋势，官方没有指定参考资料，可以看一下考上的学长学姐推荐的书目：

    • 贾俊平《统计学》：加上习题和圣才的习题详解，过两遍以上，夯实基础；
    • 茆诗松《概率论与数理统计》：书和习题必须过两遍以上，整理好笔记，重中之重，有好几年从中出题；
    • 人大出版 《应用回归分析》《时间序列分析》《多元统计分析》：重在统计应用，近来财大愈加重视时间序列和应用回归分析；

    以下书目作为补充：

    • 李贤平 《概率论基础》：加上配套的数理统计和概率论部分习题，有利于增强对概率论的理解和统计学理论知识的拓展
    • 陈希孺 《概率论与数理统计》：与茆书互为补充，部分习题很有拓展价值；
    • 统计推断：中山指定教材，看看定理和讲解即可

    此外，上财的复试也与众不同，加入了编程的考核，所以没有基础的同学在复试之前一定要多加练习这块内容，笔试的话和初试相似，需要重拾初试的笔记深入看一下，基本就是茆诗松书上的内容，比较推荐的有以下几个资料：

    • 廖雪峰 python教程
    • 菜鸟编程python100例
    • 剑指offer或者leetcode部分编程题

    从已有的就业分布来看，不管是A组还是B组的同学都会有一部分去往金融行业，大多是证券和基金等，工作岗位大部分偏向于金融领域的技术岗，另外一小部分因为之前的实习方向及取得的证书原因就纯粹在搞金融了，比如在券商做行业研究员、投行狗等。而432应用统计更加广泛的就业去向其实是在互联网行业，同时在这个行业的认可度也在稳步提高，每年都会有很大一群人去往BAT等顶尖的互联网公司。

    从职位分布上来看，去往互联网公司的主要分为两部分，大部分童鞋会去往数据分析师岗位，岗位职责在不同公司会略有不同，但都偏向于与业务衔接的数据支持以及数据分析支持，另外一部分则会去往算法工程师岗位，主要岗位职责则偏向于一部分业务衔接与更多的业界前沿方法与理论的探索，算法工程师又会分包括图像处理、文本挖掘、信息推荐等各个专业方向，看到时候自身希望以及适合的发展方向，算法会比数据分析更累一点，相应的工资也会高一个量级。去往金融公司的同学的话，则很大一部分同学都会选择数据分析岗位，工资比互联网会略低，但也比较可观，另一部分选择金融岗包括去往证券基金的同学则会有不太一样的道路发展，基本选择这条路的同学都是需要更清晰的规划与更多的实习。

    从就业地点来看，去往金融公司的基本都会留在上海，去往互联网的则会根据公司的地点选择有更多的不同，包括北京、深圳、杭州等地。

    总的来看，在上财统计学院的毕业生不仅在互联网行业越来越受认可，在金融行业也很受青睐。当然，未来选择的道路其实还是与自己期望有很大关系的，任何一条道路都会有一个好的回报，肯定不会辜负自己的努力。但你现在要做的还是踏踏实实地学习，争取明年加入到上财应用统计的大家庭中来。
