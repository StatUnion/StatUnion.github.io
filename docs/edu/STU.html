    # 小统侃院校

    上海交大，魔都王牌院校，强势985双一流，应用统计设立在数学科学学院底下，其统计学科在2012年 (第三轮) 在全国统计学一级学科评估的88家参评单位中总分排名第八，虽然第四轮的评估结果只取得了B的成绩，但依托于在全国的知名程度和优秀的师资力量，每年依然吸引了众多考研同学报考，当然其背后还有一个原因，就是上交应统连续几年扩招，17年8个、18年13个、19年蹿升到了39个、20、21年保持稳定，分别是45、47、44人，官方的说法是因学科发展需要，按照这个趋势，之后几年至少不会缩招。

    看一下上交应统的历年复试分数线，16年、17年都是355分、18年375分、19年380分、20年395分、21年400分、22年401分，虽然有逐渐上涨的趋势，但比较缓慢，相较于400+的院校来讲已经很善良了。19年进入复试的一共有49人，录取了39人，刷了10人；20年进入复试48人，录取45人，仅刷了3个人；21年65人进复试，录取47人，刷了18人；22年55人进复试，录取44人，刷了11人。就分数线和刷人比例来讲，上海交大的性价比还是比较高的。

    再看一下研究生课程的设置，专业基础课，回归分析与线性模型、应用随机过程、统计推断与决策、应用多元统计分析，都是比较硬核的统计课程，选修课中不乏金融、人工智能相关的课程，比如数理金融、最优化、大数据分析、统计学习等，课程安排多元化且深入，大家可以搭配不同的方案来适应自己的发展方向。毕业之后大家有去金融行业、咨询公司，也有前往互联网公司做数据分析、算法，各个院校的应统毕业生就业方向大同小异，基本就是上述几个方向。

    今年之前上交的专业课相较于清人复来讲相当友好，初试的参考书目只有贾俊平《统计学》第五版这一本书，茆诗松的《概率论与数理统计》也只是作为扩展延伸，据以往学长学姐的经验，茆书对于上交应统的初试来讲太难了，所以专业课复习基本内容以贾书为主，加之茆书一到四章，其中难点应该在茆书的一到四章，但是要把重心放在贾书部分。

    到了今年，最大的一个改革就是参考书改为何书元《概率论》、韦来生《数理统计》、贾俊平《统计学》，把之前官网上的《概率论与数理统计》拆分为概率论和数理统计两个部分。可以看到，明显加大了概率论和数理统计的权重，这从交大的历年真题的难度也可以看出来，小统去年就说过：“据历年真题来看，上交的真题一年难于一年，估计之后的趋势也是会加入更多的概率论数理统计的计算证明题型，毕竟几个应统王牌学校已经走在了前面，上交也必将迎头赶上。”所以今年报考上交的同学一定要铆足劲把概率论和数理统计部分给重视起来。

    以下为前几年专业课考题解析与备考方案，今年的备考大家可以根据参考书自行调整：

    1）选择题:一共30道，每题2分，其中有5道考察概率论的知识。难度中等及以下，重点考察统计学的前半册内容，描述统计的考点居多，因此大家复习前面6章时应事无巨细地不放过每一个知识点，反复记忆，达到自己看到这个专有名词就能很快反映出其意义与有关运用的效果，例如配额抽样的适用情况、雷达图的作用、置信水平为95%的意义、两类错误的定义与计算等。

    练习其他高校统计学的真题及模拟题也是十分必要的，因为选择题的考点较广泛，大家在做其他学校的题目时，可以了解到同一知识点的不同考法，有的题目很有综合性，计算量堪比大题，例如某校的一道选择题让辨别出几个统计量中均方误差最小的那一个，答案解析就有大半页A4纸，难度较高，这样的选择题就应予以重视，考前一个月把错过的选择题与正确解法拿出来复习一遍。就算有不重合的考点，也可以锻炼自己的思维能力与做题速度。

    2）简答题：一共4道，每题10分，其中有1道考察概率论的中心极限定理。简答题考察的范围较固定，主要集中在一元线性回归、多元线性回归与时间序列这三章，另外历年真题中有一类考察样本量的确定的题型，出现在第7章参数估计的第4节。大家可以从搜集到的历年真题中找到所有考点，可以到考研文库网站上搜索，一般上一年出现的几个考点应作为今年的次重点，为你排除了不太可能考的几处知识点。

    我总结出了全部考点为以下几类：误差项的正态性假定、样本量的确定、多元回归模型、多重共线性、变量选择与逐步回归、年度折叠图的画法及作用、指数平滑法、季节指数与确定并分离季节成分、时间序列要素与平稳非平稳序列的定义、时间序列预测方法的评估、两类错误的定义与表达式、中心极限定理。

    这些考点在书上对应的章节小标题处均可找到答案，表述追求完整、专业，建议大家将这些知识点的有关段落全部理解与熟练背诵，大部分知识点涉及的篇幅均不短，考试时写在答题纸上几乎会占据1/2至5/6的A4纸版面，希望大家尽可能原版还原课本的表述，展开来写而切忌三言两语。

    只有中心极限定理的那一题较灵活，一般没有标准答案，题目会给出具体的分布，一般的模式是在大量重复抽样的情况下问某统计量极限分布，大家只需要计算出该统计量的均值与方差，就可以给出极限分布-正态分布的表达式了。

    3）计算题：一共3道，前两题每题20分，最后1题10分，固定为考察概率论的内容。前两题的难度适中，认真做完课后习题应该足以应对，主要考察范围集中在假设检验与区间估计、方差分析、一元线性回归、多元线性回归这几章，其中假设检验与区间估计是必考题，大家需要熟练记忆与运用在不同情况下的统计量表达式，例如总体方差已知与未知、大样本情况与小样本情况下的运用z统计量或t统计量，t统计量的自由度该如何计算等，这些都可以在书上找到结论，有规律可循，不必死记硬背，常把公式拿出来温习巩固，不要忽略每个字母符号的细节，例如在一元线性回归中，因变量的置信区间与预测区间表达式较复杂，可以把它拆分成三个部分，t分位数、估计标准误Se与根号修正部分，总结出适合自己的记忆方法。

    方差分析的题目重视自由度与SST、SSA、SSE的准确计算，考试时当样本量多的情况下一般会给出部分计算结果，善于利用他们，能免去很多不必要的计算的麻烦。例如给出了总误差平方和SST与判定系数后，要求计算SSE，可以直接通过得到。注意计算仔细，保持冷静地使用计算器，第一次计算时不必要瞻前顾后，顺利得出结果后，有多余的时间再去检查验证，一般时间是充足的，更应保证计算不失误。

    最后1题按照近几年难度变化趋势来看，2020届考研的难度应该不会减小，是一道有区分度的题目，建议大家平时复习有余力的时候多练习茆诗松概统教材第二版的课后习题，挑一些思考量大的题目锻炼思维，这也是对数学三概统部分的拓展与强化。

    总的来说，对于不同目标的同学，在这里我提出如下几点复习方案：

    1、目标定在120分的同学可以尽全力争取贾俊平《统计学》所涵盖的130分，直接放弃那本较难的概率论与数理统计，因为这本书学习起来费时，分少，性价比低，不如将时间用在不足自己其他学科的短板上（如果其他的学科准备的都比较充分，那么这本书还是不要放弃）。

    2、目标定在130分至140分之间的同学，在完成第一类同学建议任务的基础上，再深入地将茆诗松的概率统计书中的一些重要的定理以及计算题的题型学习一下。争取能够把选择题拿到全分。

    3、目标定在140分以上的同学，由于上海交大的专业课有一定的压分现象，因此想要达到140分以上真的有一定的难度（不过不用太担心，统计学由于学科的特殊性，大部分的题对就是对错就是错，你的能力与分数时能够成正比的）需要达到140分以上只能够巨细靡遗的将两本推荐的参考书中每一个概念、定理、推论，仔细地掌握。我个人觉得，冲击140+的策略性价比较低。
