    # 小统侃院校

    私以为，北师大可能是应统考研性价比最高的院校了，教育部最新一轮学科评估排名A-，全国第7，师范类院校的top1，2019QS世界大学学科排行全球200强，2019软科中国最好学科排行前5。北京珠海两地招生，最后发的毕业证完全一样（不会有珠海两字），最关键的是，此乃招生和扩招狂魔，20年珠海校区第一年招生，报名人数310人左右，最后录取104人，报录比3：1；2021年报名700+，最后加上临时扩招和推免人数未推满，一共录取统考生144人（含1人专项计划），报录比5.6：1。

    三个方向，01经济与金融统计/数据科学与管理、02经济与金融统计/数据科学与管理（珠海）、03教育测量与大数据挖掘（珠海），三个方向分开报名、分开排名、统一划线，01跟02上课除了校区不一样，上课内容基本一致，基本上是同样的老师上半学期在北京，下半学期在珠海，03方向由统计学院和创新协同中心共同培养，在选课上与01、02方向有个别专业课不一致。

    16-20年题型较为固定，贾俊平《统计学》为主，茆诗松《概率论与数理统计》为辅。每年约有40分以上的原题或改编题，对茆书考察较为简单，多为课后习题原题或者年年考的重复题。

    21、22年题型有所改变，增加了茆书占比，删去了大部分与往年重复的题，以往真题重复考的知识点今年一个也没考，题目更灵活。难度有所提升，对数理知识的掌握显著提高，参数估计、假设检验仍是考察重点，风格更偏向数理化。这也提醒今年考生准备一定要全面，往年没有考过的知识点可能今年会出现，不要抱有侥幸心理，不要仅仅局限于贾书，茆书的地位正一步一步提升。

    关于专业课，茆诗松概率论从第三章开始，需要有一定的高数基础，个人建议先学完数三高数部分，如果概率论基础为0，可以再学完数三范围内的概率论再学习茆诗松的概率论，第一章中的基本模型需要全部掌握，课后习题可以进行分类，部分题目解法相同，掌握解题思想即可，第二三章是全书重点，需要全部掌握，第四章难度偏大，如果觉得困难，可以暂时跳过，第五章推导内容较多，建议自己推导一遍，第六章重点为1，2，6节，第6节区间估计和第七章假设检验可以互相转换，第七章学习重点为前3节。前七章每个部分至少看三遍以上，重点部分不能低于5遍，所以当你第一遍学习时有些不明所以也正常，只要你潜下心来仔细钻研，最终应该都能明白。

    可以看下19年北师大的就业去向（19年珠海校区未开始招生），19届应用统计专硕毕业71人，70%的同学留在了北京，考北师大还有个优势是，不管你是什么专业，都可以转行当老师：

    • 去往金融35% 互联网31% 政府及事业单位24% 其他15%
      • 互联网：华为、滴滴、京东、字节跳动
      • 金融：中国工商银行总行、中国农业银行、招商银行分行、中国银行分行
      • 政府及事业单位：国家烟草专卖局、中国电信研究院

    北京校区坐落于北京三环内，虽然不提供住宿，但是实习机会遍地都是，可以逃课在研一下就开始实习。

    珠海校区坐落在美丽的珠三角，占地面积5000亩！在凤凰山风景区内，校内随处一拍都是风景，校内大到有时候上课需要坐观光车，每天步数都是1W起步。“亚洲最美山谷大学”。不管离深圳，广州都很近，还有港珠澳大桥，是不是周末就去澳门玩一圈。珠海有很多小岛，周末时不时可以去岛上玩一圈。

    所在城市 北京/珠海
    办学水平 985

    考试课目 政治、英二、数三、432统计学

    3.教育测量与大数据挖掘（珠海）
