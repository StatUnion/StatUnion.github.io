    # 小统侃院校

    复旦，偶像级院校，在金融法和数学方向独占鳌头，15年起在数学科学学院增设应用统计专业硕士学位，同年复旦大学成立大数据学院，也设置了应统专业。此外，数学中心也会招收少量的学生。

    数学科学学院的课程编排大部分为统计课程，包含数理统计，随机过程，时间序列分析，正则化与机器学习等，其余为金融类课程，精算学、投资学、金融衍生工具，总体来说，更偏金融统计方向，数院毕业的学生大多会去券商基金做金融相关的工作，学院的传统如此，也有少部分会去互联网从事算法工作；录取名额有逐年增加之势，19年3人，20年9人，21年18人，22年10人。

    大数据学院则把重心放在了大数据上，开设大数据算法、时间序列与空间统计、人工智能与机器学习、神经网络和深度学习、最优化理论、数据挖掘等人工智能课程，网上盛传的《神经网络和深度学习》书作者邱锡鹏老师就在大数据学院讲授这门课，总的来说重算法和计算机，大多数学生会去互联网公司从事算法工作或者去金融行业觅得一职。每年招生的人数可谓充沛，19年进复试20人，录取了20人，进了复试就是复旦人；20年进复试48人，录取43人，刷了5人；21年进复试77人，录取了62人，淘汰15人；22年进复试67人，录取了55人，淘汰12人。

    数学中心的课程设置可以参考数学科学学院，两者的研究方向也是一样的。2020年是上海数学中心第一年招收应统统考生，计划人数15人（不招收推免生），上线12人，最终录取10人，复试线370分。21年仅录取4人。22年录取10人，仅有1人统考，其余九人都是调剂进入的。

    学费异常昂贵，一年10.9万，而且不提供住宿，不过会提供校外学生公寓，一般是四人间，1300元/月，此外还会有交通补助费，800元/月。这样的学习费用确实会劝退一大波人，复旦虽好，但学费也不至于高的那么离谱。这样相比来看，北京的一些院校在学费上要良心的多了。不过即使有这么高的学费门槛，每年报考复旦应统的同学还是很多，谁让他是复旦呢。

    关于专业课，这三个学院是统一命题的，题型主要是计算题与证明题，2021年及2017年以前曾考过名词解释，整体以计算证明为主，题量通常在8-12题，重点考察概率论部分且每年分值最多、题目较难，数理统计考察相对简单。

    强烈推荐茆诗松《概率论与数理统计》和李贤平《概率论基础》。这几年真题大多可以从这两本中找到原题。茆诗松教材逻辑清晰，难度适中，面面俱到；李贤平教材针对各个专题设计了挺有深度的题目。这两本书可以说是非常互补的。

    建议前期先看茆诗松《概率论与数理统计》打好基础，从头到尾看一遍，包括定理，完成所有课后习题，不会的记得标注，到时候二刷的时候可以更顺手一些。实际上这本书看完你已经可以拿到不错的分数；再去看李贤平概率论基础增强你在概率论方面尤其是收敛性的知识，注意李书中茆诗松没有的题型和专题，很容易考。李书里太难的题目可以直接跳过，考场上几乎没有人能做出来，这两本书建议多刷几遍，难题证明题反复做，可以自己总结专题题型，以便考前复习，题刷多了你就会发现你做题的手感越来越好。

    后期的复习以复旦真题为主，对自己要求高的同学可以搭配北大应用统计、清华应用统计、北大金融数学的真题进行训练，这些学校的出题风格都较为类似，但是对于报考复旦大学应用统计的同学来说，回归分析和方差分析不是非常必要去做太多题。

    复试根据不同学院科目内容不同，数院和数学中心应用统计复试内容主要是数学分析、高等代数、概率论与数理统计，大数据学院复试内容相对更广泛，包含数学分析、高等代数、概率论与数理统计、数据结构与算法、机器学习、数据库等。
