    # 小统侃院校

    作为全国top2院校的清华，其地位与资源自不必说，竞争相当激烈，难度也是属于前列。清华应统设立在数学科学系下，学制两年，没有具体的研究方向，第一年不设导师，纯上课，第二年选导师。报考清华并上岸的同学一般本科基础也很好，也不乏二战三战的。

    清华应统初试没有大纲和参考资料，几年招生下来，根据学长学姐的总结就是，偏重数理统计，什么都有可能考，证明题出题非常变态。一般是5-6道大题的形式。近几年真题总体风格趋向于基础加创新,注重对基础知识的考查，尤其是常见分布的各种变换,例如指数分布、卡方分布、均匀分布、贝塔分布等，分布间的各种变换是需要学生熟练掌握，这也是整场考试的基础。其次是一些基础知识点，例如充分完备统计量,无偏估计，相合估计，最短置信区间的构造，p值，似然比检验等。基础部分占到4/5左右。最后是创新，每年都会有新颖的知识点，这就考查学生的全面性与灵活性，例如：利用均匀分布随机模拟，多元统计矩阵分解，交叉核验，简单马尔可夫链等。且这部分风格各年均有差异，这是报考清华所要承担的风险之一。2022年真题继承了近几年风格且难度上趋于简单，计算量上也有所减少，但也涉及了一些与之不同的知识点，例如联合置信域。

    初试需要看的书很多：

    必看：

    • 概率论与数理统计教程（第二版） 茆诗松：这是参考书里难度最低的一本，考查重点，一定要把除了第八章之外的内容全部掌握，起码要刷三遍以上。
    • 概率论基础（第三版）李贤平：这本书是复旦数学系的教材，看的时候可以选择性的放弃理解定理证明，比如泊松过程之类的，直接记住结论，但是其他的习题一定要掌握，重点补充里面的强大数定律和博雷尔康特立引理，这是茆诗松的书里没有涉及到的，重点补充第五章“极限定理”
    • 数理统计（第二版）韦来生

    选看

    • 统计推断（第二版，中译本）Casella：一本管理类的研究生统计学教材，里面涉及到了一些比较高级的知识点，在初试的时候会比陈家鼎的《数理统计学讲义》更实用一些
    • 数理统计学讲义（第三版）陈家鼎：可以做一下第一章和第二章，其他部分几乎不可能考。
    • 高等数理统计（第二版）茆诗松

    温馨提示：本科数学系的同学萌请好好学习实变函数。

    强烈推荐B站“概率统计小迷哥”的视频，对于一些难以理解的概念还有如何使用茆诗松《概率论与数理统计教程》这本书的讲解都很细致且用心，应统联盟也有开设小迷哥的课程，可以作为参考。

    最后两个月一定要多刷真题，真题是提高综合能力最快捷的方法，有条件的话就把能买到的题都刷一遍，清北复人和中科大的真题都值得一做。
