    # 小统侃院校

    人大的应统设在统计学院下，学费是每年2w5，包住宿，招生人数还是比较良心的，每年都是40+或者50+（不包括保研），此外保研的名额也很充沛，每年有20+个。报名的童鞋人数保持在400+，其中二战考上的同学还是蛮多的，外经贸、中财、上财、西财等财经类院校包揽了很多的席位。

    人大应统分三个方向，分别是大数据、精算和应用统计方向。

    大数据方向课程设置上有linux操作系统、python编程、数据库、机器学习、文本挖掘、深度学习、分布式计算这些与人工智能相关的课程，比较全面合理，基本学完了就可以去公司做数据分析或者算法的实习了（算法需要自己再学下数据结构与算法，刷下题等）。每年会从考上的同学中选拔10~15位，选拔会在开学之后的两周之内进行，主要会考察每个人的和数据分析挖掘相关的项目经验，如果你是CS出身，必然也是奔着这个专业去的，非常有优势，非CS出身的同学可以在暑假的时候积累一些项目的经验，学一下python和SQL，打一下kaggle比赛、天池比赛等等。毕业之后大部分都去了互联网公司从事数据分析或者算法的工作。

    应统精算方向的复试和其他两个方向单独分开，专门考察精算方面的知识，基本上选这个方向都是考研之前就决定好了，每年大致有5~10左右的精算同学。大家会在研究生期间考一个北美精算师或者中国精算师，前者难度大一些。课程设置也多和精算挂钩。人大精算是精算院校中的老牌且实力雄厚，其保险精算中心会组织北美精算师（SOA）的考试，这个方向毕业了基本会去保险公司、银行等。

    应统方向的课程主要还是对考研时学的那几门课程的深化，像时间序列、多元统计这些都会高阶的学一些理论和应用，也会有不少童鞋会选修大数据方向的课程，技多不压身。一届的人数大约为30左右，应统方向的童鞋会朝着互联网的数据分析师、算法，政府机构、银行金融机构职员的方向发展。

    人大应统专硕从11年开始招生，前几年的考题难度和分数线相当亲民，考察的题型偏应用，基本上过几遍贾俊平的《统计学》专业课就可以考的不错了，但近几年风格在逐年的发生变化，增加了50分左右的计算证明题，大家在复习的时候要多加留心几本书的证明计算，尤其是多元和回归。

    # 学院简介

    中国人民大学统计学院始建于1952年，是国内最顶尖的统计学科平台之一，在教育部第四轮一级学科评估中，中国人民大学统计学科位列全国第一。研究生招考专业有三个：统计学、应用统计、流行病与卫生统计学。初试具体考试科目如下表所示：

    报考专业 科目一 科目二 科目三 科目四
    统计学 政治 英语一 数学二 统计学805
    应用统计 政治 英语一 数学三 统计学432
    流行病与卫生统计学 政治 英语 卫生统计613

    2015年，中国人民大学统计与大数据研究院成立，并于2020年开始正式招收推免和统考研究生（应用统计专业学位，但与统计学院的应统专硕不同）与博士生。人大统计学院有若干相互支撑的重要研究平台（一共拥有12个科研中心），强调扎实的统计理论方法和与其他学科交叉发展的强大统计应用，师资力量雄厚。当前，学院的学科涵盖数理统计、社会经济统计、生物卫生统计、风险管理与精算、数据科学与大数据技术等多个子学科，是全国拥有子学科最全的统计学院，也是最早和唯一拥有理学、经济学、医学三大学科门类的统计学院。

    # 复试介绍

    作为34所自划线院校之一，人大应统的复试时间一般较早，安排在每年3月中上旬左右（2020年特殊情况除外）。一般分为笔试与面试两部分。笔试150分，分为外语笔试（50分）与专业课笔试（100分）。面试分为英语能力面试、专业能力面试与综合素质面试。

    笔试时长3小时，外语笔试一般是听力+中译英+英译中，多半是政府报告的翻译，建议多背一背政策、经济方面的名词。专业课笔试与初试的形式类似，但是建议进入复试的同学除过最基本的几个专题，再多复习一些抽样技术方面的知识内容。20/21年有些不同，由于疫情原因，今年的线上复试笔试部分仅有45分钟（专业课30分钟，英语15分钟），专业课一共三道大题，外语为五道英译中。

    面试时长一共30分钟左右，最基本的是准备中英文自我介绍，包括自己本科所修课程、科研能力等介绍。外语面试大约10分钟，除了自我介绍之外，老师一般还会自由提问几个问题，最后一般会有短文朗读，要求朗读一段给出的与专业相关的短文，然后回答老师所提出与短文相关的问题。综合素质面试与专业能力面试在一起进行，大约20分钟，一般先由考生抽取2道专业课题目，每道题回答后老师会再延伸性地问几个问题，综合素质方面，建议考生充分了解自己的毕业设计、做过的科研竞赛等等细节问题，以备老师提问。

    # 学费学制

    人大应统的专业硕士学制为2年，每年学费为2.5w左右。学校提供住宿，宿舍一般为四人间或者六人间。

    # 研究方向

    人大应统专硕在入学之前的考察不区分研究方向，入学后会有三个方向。

    大数据方向：每年入学后两周之内会从录取的同学中选拔10~15位，建立大数据班进行培养。主要学习Linux操作系统、Python编程、数据库、机器学习、文本挖掘等等相关的课程。毕业之后大多从事互联网公司的数据分析或者算法方面的工作。没有入选的同学也无需担心，可以同时选修大数据班的课程。

    风险精算方向：每年大概5~10名左右，课程设置多与精算挂钩。人大精算是精算院校中的老牌，实力雄厚，其保险精算中心会组织北美精算师(SOA)的考试，毕业之后大多从事保险公司、银行方面的工作。

    应用统计方向：人数大概30人左右，课程设置主要是对考研时学的那几门课程的深化，比如时间序列、多元统计等课程的高阶理论和应用。毕业方向一般是互联网数据分析师、算法工程师、政府机构、银行金融机构职员等等。

    各个研究方向的具体培养方案请见如下网址：「http://stat.ruc.edu.cn/more.php?cid=89334」

    # 导师选择

    人大一般都是在入学之后填写志愿选择导师，但是建议有把握的同学在初试成绩或复试成绩公布后就抓紧联系导师，在人大统院官网查找，或通过其他途径了解各个导师的资料，选择与自己感兴趣的研究方向符合的导师，发送自荐信、推荐信、个人资历证明材料等到导师的邮箱，等待导师回复。

    这里着重建议，对于应用统计专硕的同学，选择导师不能只看导师的科研方向与科研能力。专硕的同学以后的去向大多为就业，应提前与自己心仪的导师沟通清楚，其研究重点是方法上的研究还是应用型研究，其研究强度是否适合学硕而非专硕，否则可能会对之后的学习、实习的时间安排造成较大影响。最直接的方法，可以去找一两篇心仪导师的论文来看，基本能看出其研究重点与难度是否适合自己。

    人大统院导师的研究方向有大数据方向、社会统计、数理统计、风险精算与生物统计等等，具体可通过如下官网网址进行查看。「http://stat.ruc.edu.cn/teacher.php?cid=25」
